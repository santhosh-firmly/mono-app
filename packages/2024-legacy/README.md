# Drop-in Service

## Running the storybook

### Pre requisites

1. Make sure you have node.js installed (v16+). You can use a node.js version manager for this.
2. Make sure you set up the dependencies with `npm ci`.

### Running locally

1. Execute the story book with `npm run storybook`.
2. If it does not automatically open the server. Go to your browser and open http://localhost:6006
