# Typography

The Firmly Dashboard uses **Inter** as the primary font family, with **Josefin Sans** for specific branding elements like the Firmly logo.

## Font Families

### Inter

Inter is a highly readable typeface designed for computer screens. It's used throughout the dashboard for all UI text.

<div>
  <p style={{ fontFamily: 'Inter, sans-serif', fontSize: '24px', fontWeight: 400 }}>Inter Regular - The quick brown fox jumps over the lazy dog</p>

  <p style={{ fontFamily: 'Inter, sans-serif', fontSize: '24px', fontWeight: 700 }}>Inter Bold - The quick brown fox jumps over the lazy dog</p>
</div>

### Josefin Sans

Used exclusively for the Firmly brand name and logo.

## Font Weights

| Weight | Class | Usage |
|--------|-------|-------|
| 400 | `font-normal` | Body text, descriptions |
| 500 | `font-medium` | Labels, nav items, emphasis |
| 600 | `font-semibold` | Headings, buttons |
| 700 | `font-bold` | Page titles, strong emphasis |

## Best Practices

### Hierarchy

1. Use **one primary heading** per section (usually the page title)
2. Use progressively smaller headings for nested sections
3. Don't skip heading levels (h1 â†’ h3)

### Readability

- Keep line length between 50-75 characters for optimal readability
- Use `tracking-tight` for headings to improve visual weight
- Add sufficient line height for body text

### Consistency

- Use `font-semibold` for all headings (not bold)
- Use `font-medium` for labels and navigation
- Use `font-normal` for body text

### Accessibility

- Maintain minimum 16px for body text
- Ensure sufficient color contrast
- Don't rely on text styling alone to convey meaning
