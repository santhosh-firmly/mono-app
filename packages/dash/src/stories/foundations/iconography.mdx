# Iconography

The Firmly Dashboard uses **Lucide Icons** for consistent, clean iconography throughout the interface. Lucide is a fork of Feather Icons with additional icons and active maintenance.

## Installation

Icons are imported from `lucide-svelte`:

```svelte
<script>
  import Home from 'lucide-svelte/icons/home';
  import Settings from 'lucide-svelte/icons/settings';
  import User from 'lucide-svelte/icons/user';
</script>

<Home class="h-5 w-5" />
```

---

## Icon Sizes

We use three standard icon sizes throughout the dashboard:

| Size | Class | Usage |
|------|-------|-------|
| 16px | `h-4 w-4` | Inline with text, badges, compact buttons |
| 20px | `h-5 w-5` | Navigation, button icons, form fields |
| 24px | `h-6 w-6` | Headers, empty states, large buttons |

### Size Examples

```svelte
<!-- Small (16px) - inline icons -->
<span class="flex items-center gap-1">
  <Check class="h-4 w-4" />
  Verified
</span>

<!-- Medium (20px) - buttons and nav -->
<button class="flex items-center gap-2">
  <Plus class="h-5 w-5" />
  Add Item
</button>

<!-- Large (24px) - headers and empty states -->
<div class="flex flex-col items-center gap-4">
  <Package class="h-6 w-6 text-muted-foreground" />
  <p>No orders yet</p>
</div>
```

## Icon Colors

Icons inherit the text color by default. Use Tailwind classes to control color:

```svelte
<!-- Default (inherits text color) -->
<Settings class="h-5 w-5" />

<!-- Muted -->
<Settings class="h-5 w-5 text-muted-foreground" />

<!-- Success -->
<CheckCircle class="h-5 w-5 text-green-600" />

<!-- Error -->
<AlertCircle class="h-5 w-5 text-red-600" />

<!-- Warning -->
<AlertTriangle class="h-5 w-5 text-yellow-600" />

<!-- Info -->
<Info class="h-5 w-5 text-blue-600" />
```

## Best Practices

### Consistency

1. **Use standard sizes** - Stick to h-4, h-5, or h-6
2. **Align with text** - Use `flex items-center gap-2` for icon + text
3. **Match icon weight** - Lucide icons have a 2px stroke weight

### Accessibility

1. **Add labels** - Icons used as buttons need aria-label
2. **Don't rely on icons alone** - Pair with text when meaning isn't obvious
3. **Ensure contrast** - Icon color should have sufficient contrast

```svelte
<!-- Icon-only button with label -->
<button aria-label="Delete item">
  <Trash2 class="h-5 w-5" />
</button>

<!-- Icon with visible text -->
<button>
  <Trash2 class="h-5 w-5" />
  <span>Delete</span>
</button>
```

### Loading States

Use animated icons for loading states:

```svelte
<script>
  import Loader2 from 'lucide-svelte/icons/loader-2';
</script>

<Loader2 class="h-5 w-5 animate-spin" />
```
