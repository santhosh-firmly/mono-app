The Firmly Dashboard uses Tailwind CSS's default spacing scale, which provides a consistent rhythm throughout the interface.

## Spacing Scale

## Common Spacing Patterns

### Card Padding

Cards use consistent padding for their content areas:

```html
<!-- Standard card -->
<div class="p-6">...</div>

<!-- Compact card -->
<div class="p-4">...</div>

<!-- Header with bottom padding -->
<div class="pb-4">Title</div>
```

### Form Layouts

Form elements follow a predictable spacing pattern:

```html
<!-- Form field group -->
<div class="space-y-2">
  <label>Email</label>
  <input />
</div>

<!-- Between form fields -->
<div class="space-y-4">
  <div>Field 1</div>
  <div>Field 2</div>
</div>

<!-- Form sections -->
<div class="space-y-6">
  <section>Section 1</section>
  <section>Section 2</section>
</div>
```

### Grid Gaps

Grid and flex containers use consistent gap values:

```html
<!-- Tight grid (icons, chips) -->
<div class="gap-2">...</div>

<!-- Standard grid (cards, items) -->
<div class="gap-4">...</div>

<!-- Wide grid (page sections) -->
<div class="gap-6">...</div>
```

## Page Layout Spacing

### Container Padding

```html
<!-- Mobile -->
<div class="px-4">...</div>

<!-- Desktop -->
<div class="px-6 lg:px-8">...</div>
```

### Section Spacing

```html
<!-- Between page sections -->
<div class="space-y-6">
  <section>...</section>
  <section>...</section>
</div>

<!-- Large section breaks -->
<div class="space-y-8">
  <section>...</section>
  <section>...</section>
</div>
```

### Sticky Elements

```html
<!-- Sticky header -->
<header class="sticky top-0 z-50 py-4">...</header>

<!-- Sticky footer button -->
<div class="fixed bottom-0 p-4">...</div>
```

## Best Practices

### Consistency

1. **Use the scale** - Stick to the defined spacing values
2. **Group related elements** - Use smaller spacing within groups, larger between
3. **Maintain rhythm** - Use consistent spacing throughout the interface

### Responsive Spacing

```html
<!-- Responsive padding -->
<div class="p-4 md:p-6 lg:p-8">...</div>

<!-- Responsive gaps -->
<div class="gap-4 md:gap-6">...</div>
```

### Avoid

- Arbitrary values like `p-[17px]`
- Inconsistent spacing between similar elements
- Too much spacing (creates disconnect)
- Too little spacing (feels cramped)
